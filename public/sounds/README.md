## Sound Sources

Current alarm sounds are sourced from Freesound.org under Creative Commons licenses:
- Some sounds may be previews (low-quality versions)
- For production use, consider downloading full-quality versions
- Always check and comply with attribution requirements

## Format Requirements

- Format: MP3 (or OGG for wider browser support)
- Sample rate: 24-48 kHz recommended
- Bitrate: 64-192 kbps
- Duration: 2-10 seconds (the audio manager supports looping)
