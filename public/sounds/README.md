# Alarm Sounds

Place your alarm sound files here.

## Required Files

- `alarm-1.mp3` - Classic Alarm
- `alarm-2.mp3` - Beep Beep
- `alarm-3.mp3` - Rooster
- `alarm-4.mp3` - Bell
- `alarm-5.mp3` - Chime
- `alarm-6.mp3` - Digital

## Where to get alarm sounds

You can find free alarm sounds at:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/alarm/
- https://www.zapsplat.com/

## Format

- Format: MP3
- Sample rate: 44.1 kHz or 48 kHz
- Bitrate: 128 kbps or higher
- Duration: 3-10 seconds (can loop)
